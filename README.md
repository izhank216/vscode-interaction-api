# vscode-interaction-api

Unofficial interaction API for VSCode.

---

## Installation

Install via npm:


`npm install @izhank216/vscode-interaction-api`

Or with Yarn:

`yarn add @izhank216/vscode-interaction-api`

## Build from source
`git clone https://github.com/izhank216/vscode-interaction-api.git`

Then install dependencies:

`yarn install`

Then compile it:

`yarn build`
